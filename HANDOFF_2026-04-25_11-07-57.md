## Decisions Made

- **Orange star uses 🟠 emoji** (orange circle) in the results screen, and a hue-rotated ⭐ emoji in the header counter — this gives a distinct orange star appearance without needing custom SVGs.
- **Mistakes persist globally** in localStorage alongside total stars — they accumulate across all sessions, same as gold stars.
- **All game modes** trigger orange stars via the shared `onWrong` callback in `app.js:startGame()` — no per-mode changes needed.
- **Memory game timer** starts on "Start!" click, not when cards are first shown — preview/memorization time is free.
- **Header layout**: Both counters are wrapped in `.counters-group` flex container, keeping them grouped on the right side of the header bar.

## Direction of Development

This is a kid-friendly Danish multiplication tables practice app with 7 game modes:
1. **Udfyld Svaret** (Fill in the Blank) — type the answer
2. **Vælg det Rigtige Svar** (Multiple Choice) — pick from 4 options
3. **Find den Manglende Faktor** (Missing Factor) — find the missing number
4. **Tidsudfordring** (Speed Challenge) — 60-second timed mode
5. **Hukommelsesspil** (Memory Game) — match expressions with answers
6. **Hele Tabellen** (Table Sequence) — fill in missing values in a row
7. **Lang Division** (Long Division) — step-by-step guided division

Architecture: Single-page vanilla JS app (no frameworks/build tools). Game modes register themselves via `window.GameModes` and are loaded as separate script files. Shared state lives in `window.AppState`, shared logic in `window.App`. Danish TTS via Web Speech API (prefers macOS "Sara" voice). All persistence via localStorage.
